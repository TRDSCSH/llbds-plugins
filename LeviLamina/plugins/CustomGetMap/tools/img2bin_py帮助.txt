用法：

python pyimg2bin.py -f <file> -d <dir> -W <width> -H <height>

命令行参数说明：

-f <file>, --file <file>：指定输入图片文件的路径和文件名。
-d <dir>, --dir <dir>：指定输出二进制文件的文件夹路径。（可选，默认为图片名称）
-W <width>, --width <width>：可选项。指定输出地图画的宽度（方块格数）
-H <height>, --height <height>：可选项。指定输出地图画的高度。（方块格数）
示例用法：

将名为input.png的图片文件转换为二进制文件，并将输出文件放置在默认的文件夹input中：

python pyimg2bin.py -f input.png

将名为input.png的图片文件转换为二进制文件，并将输出文件放置在文件夹output中：

python pyimg2bin.py -f input.png -d output

将名为input.png的图片文件转换为二进制文件，并指定输出图片的宽度和高度为2：

python pyimg2bin.py -f input.png -W 2 -H 2
